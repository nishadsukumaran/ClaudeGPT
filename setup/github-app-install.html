ClaudeGPT — GitHub App install

Pre-fills GitHub's "Create App" form. GitHub requires a public callback URL, so we'll use smee.io as a free relay.

Step A — Open https://smee.io/new in a new tab. It auto-generates a URL like https://smee.io/abc123xyz. Keep that browser tab open — GitHub's redirect will land there. Paste the URL into the form below.
Required by GitHub. Must be HTTPS. The smee.io page will show the temp code GitHub sends after creating the App.
Pick something distinctive. GitHub rejects duplicates.
If you haven't deployed yet, leave the placeholder. You can edit it in App settings later. For local dev, you can paste a second smee.io URL.

What this App will request

Permissions: Contents (write), Issues (write), Pull requests (write), Metadata (read), Checks (read).
Events: issues, issue_comment, pull_request, pull_request_review, push.
Visibility: Private (only you can install it).

After GitHub creates the App

Step B — Switch to your smee.io tab. You'll see GitHub's GET request appear in the request log. The URL has ?code=... at the end. Click the request to expand it and copy the code query parameter.
Step C — Paste the code into the Claude chat. Claude exchanges it for App ID + private key + webhook secret and writes them into your .env.
